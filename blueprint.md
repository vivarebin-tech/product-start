
# Lotto Number Generator

## Overview

This is a simple web application that generates and displays lottery numbers.

## Implemented Features

*   **UI:**
    *   A title "Lotto Number Generator".
    *   A section to display the generated lottery numbers.
    *   A button to trigger the number generation.
*   **Functionality:**
    *   Generates 6 unique random numbers between 1 and 45.
    *   Displays the generated numbers on the screen.
